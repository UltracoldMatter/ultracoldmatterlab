Description of the Project

This website is a part of the Ultracold Matter Lab project. It is built using GitHub Pages and showcases real-time dashboards for temperature and humidity monitoring using DHT11 sensors and ESP32 microcontrollers.

The project features:

  • Live data streaming via MQTT to InfluxDB
  • Custom Grafana dashboards for real-time monitoring
  • Web interface hosted on GitHub
